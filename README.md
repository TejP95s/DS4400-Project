# DS4400-Project

## Setup
1. Clone the repo and ideally create a virtual environment.
2. Navigate to the DS4400-Project directory and run `pip install -r requirements.txt` to download the required packages
3. Go to [this dataset ](https://www.kaggle.com/datasets/austinreese/craigslist-carstrucks-data) from Kaggle and download the csv file.
4. Place the csv file in the `DS4400-Project/data` folder
